\# Shadow Fiend Gyrteru 1v1 Bot



Welcome to the \*\*Shadow Fiend Gyrteru 1v1 Bot\*\*! This bot is designed exclusively for the \*\*1v1 Solo Mid\*\* lobby mode in Dota 2. 



I decided to create this project because most of the existing SF 1v1 bots I found were either outdated or lacked in-depth, detailed logic. 



\*\*AI-Powered:\*\* The entire codebase was written using neural networks on \[arena.ai](https://arena.ai), predominantly utilizing `gemini-3.1-pro-preview` alongside various random models in Agent Mode.



\---



\## ✨ Features \& Capabilities



I focused on creating a bot that feels smart and responsive. Here is what it can do:



\*   1. \*\*Advanced Shadowraze Targeting:\*\* Highly accurate custom aim logic for all three Razes.

\*   2. \*\*Dynamic Playstyle:\*\* The bot understands threats and seamlessly switches its playstyle depending on the current game state and lane situation.

\*   3. \*\*Smart Retreat System:\*\* Knows exactly when it's time to back off and survive.

\*   4. \*\*Core Mechanics:\*\* Optimized logic for last-hitting, denying, and warding.

\*   5. \*\*Vanilla Scripts Overridden:\*\* Standard default Valve bot scripts have been completely disabled to ensure custom logic takes priority.

\*   💡 \*\*Anti-AFK Economy Fix:\*\* Fixed the issue where bots receive a massive gold spike at the 5-minute mark due to AFK teammates. The script now spawns 4 Beastmasters on team who actively farm jungle creeps for XP, preventing the game from flagging them as inactive.
